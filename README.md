# 期权盲盒项目

期权盲盒，不装大师，只卖心跳。

这是一个纯静态网页原型：点击盲盒，随机开出一张币卡和一张任务卡。它不是策略推荐器，而是一个简洁、好玩、带纪律边界的交易员刺激小游戏。

## 当前版本

- 随机币种池：BTC / ETH / SOL / HYPE / XRP / AVAX / TRX
- 完整 36 张任务卡
- 起始页为正方形黑金盲盒，点击后有开盒动效
- 结果页显示两张卡：币卡 + 任务卡
- 宽屏两卡并列，手机端两卡上下排列，并保持同样卡牌比例
- 底部文案：游戏，别上头
- 操作按钮：重新抽盲盒、分享朋友圈
- 底部包含 SignalPlus 广告和冰火岛社区打赏信息，点击打赏模块可复制 EVM 钱包地址
- 单文件静态原型，适合直接本地打开，审定后可部署到 GitHub Pages

## 全局规则

- 只用娱乐仓位
- 最大亏损提前定死
- 不加仓、不补仓、不复仇
- 亏完门票就结束
- 抽卡可以随机，仓位不能随机

## 文件

- `index.html`：页面、样式、币种池、卡牌池和抽卡逻辑都在这一份文件里，方便直接分享和部署
- `打开预览.bat`：双击即可用默认浏览器打开本地页面
